Changelog


In this new release TeXmacs for windows


  • Unicode filenames are now handled.

  • The setup now sets menu items, icons and file associations

  • The updater is now activated : a popup window will inform you of future releases at this place


The enabling of unicode filenames required many changes which may have introduced bugs ; this release should therefore be regarded as a beta version. Please test unicode support and report here any issue related to the file system.
